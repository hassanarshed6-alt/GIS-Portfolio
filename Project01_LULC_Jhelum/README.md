# Project 01 – Land Use Land Cover Classification

## Study Area
Jhelum District, Punjab, Pakistan

## Objective
To classify land cover into five classes using Sentinel-2 imagery and Random Forest classification.

## Data

- Sentinel-2 Surface Reflectance
- Spatial Resolution: 10 m
- Date: October 2025

## Software Used

- Google Earth Engine
- QGIS

## Classification Method

- Supervised Classification
- Random Forest

## Classes

- Water
- Vegetation
- Agriculture
- Built-up
- Barren Land

## Results

Overall Accuracy: 95.3%

Kappa Coefficient: 0.94

## Outputs

- LULC Map
- Classified Raster
- Accuracy Assessment
