# Land Use Land Cover Classification – Jhelum District, Pakistan

## Project Overview

This project performs supervised Land Use Land Cover (LULC) classification for Jhelum District, Pakistan using Sentinel-2 imagery in Google Earth Engine.

## Objectives

* Generate an LULC map
* Classify major land cover classes
* Assess classification accuracy
* Produce a professional cartographic layout

## Study Area

**Jhelum District, Punjab, Pakistan**

## Dataset

* Sentinel-2 Surface Reflectance (Level-2A)
* Spatial Resolution: 10 m
* Acquisition Period: October 2025

## Software Used

* Google Earth Engine
* QGIS
* ArcGIS Pro
* Python
* PostgreSQL/PostGIS

## Classification Method

* Supervised Classification
* Random Forest Classifier
* Training Sample Collection
* Accuracy Assessment using Confusion Matrix

## Land Cover Classes

* Water
* Vegetation
* Agriculture
* Built-up
* Barren Land

## Results

* **Overall Accuracy:** ~95%
* **Kappa Coefficient:** ~0.94

## Outputs

* Classified raster
* LULC map layout
* Accuracy assessment
* Area statistics
